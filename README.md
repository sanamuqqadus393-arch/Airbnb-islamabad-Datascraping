# Airbnb Islamabad Data Scraping 🏡

A Python-based web scraping project that collects Airbnb listing data from Islamabad for analysis and business insights.

## 📌 Overview
This project extracts Airbnb listing data from Islamabad including:
- Price
- Location
- Reviews
- Description

## ⚙️ Technologies Used
- Python
- BeautifulSoup
- Pandas

## ✨ Features
- Scrapes real-time Airbnb listing data
- Extracts price, location, reviews, and descriptions
- Stores structured data in CSV format
- Easy to modify for other cities

## 📊 Output
The scraped data is cleaned and stored in a structured CSV file, ready for:
- Data analysis
- Visualization
- Business decision-making

## ▶️ How to Run

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt

3. Run the scraper:
   python scraper.py

## 🚀 Use Cases
- Price analysis
- Market trends
- Airbnb business insights
